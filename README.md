# 💼 Rita Pessoa | Portfolio Website

My personal portfolio website — a multi-page site built with HTML, CSS, and JavaScript to showcase my projects, skills, and journey as a front-end developer.

**Live demo:** [ritas-portfolio.netlify.app](https://ritas-portfolio.netlify.app/)

---

## 📋 About

This is my personal corner of the web. I built it from scratch to have a professional online presence that reflects who I am as a developer: someone who cares about both clean code and good design. It features four pages: Home, About, Projects, and Contacts.

---

## ✨ Features

- 🏠 **Home** - Hero section with a short introduction and featured project preview
- 👩 **About** - My background, skills, and journey into tech
- 🗂️ **Projects** - Showcase of all my built projects with descriptions and live links
- 📬 **Contacts** - Ways to get in touch
- 📱 Responsive layout across all screen sizes
- 🔗 Open source - the code is right here!

## 🚀 How to Run Locally

No build tools required.

1. Clone the repository:
```bash
git clone https://github.com/ritapessoa01/ritas-portfolio.git
cd ritas-portfolio
```

2. Open `index.html` in your browser:
```bash
open index.html
# or just double-click the file
```
